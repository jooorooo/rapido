## Description


This library is a PHP Oriented Object client for the [rapido.bg](http://www.rapido.bg/soap_help/) SOAP API Services.


## Setup 

You can use composer to use this library.

```php
{
    "require": {
		"simexis/omniship-rapido": "*"
    }
}
```
